## Simple bot for my flatmates
### Features:
- Remind for apparments cleaning
- Hate you

## Setup
### Create & activate virtual environment
```
python3 -m venv "venv"
source venv/bin/activate
```


### Install dependencies
```
pip3 install -r requirements.txt
```

### Setup tokens in env
```
TELEGRAM_TOKEN
AIRTABLE_TOKEN
AIRTABLE_ID
```
