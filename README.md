## Simple bot for my flatmates
### Features:
- Remind for appartments cleaning
- Hate you
- Make some jokes
- Show war stats
- Show weather forecast

## Setup
### Create & activate virtual environment
```
python -m venv "venv"
source venv/bin/activate
```


### Install dependencies
```
pip install -r requirements.txt
```

### Setup tokens in env
```
TELEGRAM_TOKEN
TELEGRAM_CHAT_ID
AIRTABLE_TOKEN
AIRTABLE_ID
```

## Start
```
python bot.py
```
